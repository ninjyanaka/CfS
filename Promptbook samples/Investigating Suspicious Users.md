# 疑わしいユーザーの分析

**利用するプラグイン** : Microsoft Entra, Microsoft Intune

**入力パラメータ**: UPN 

**概要**:　ユーザーのサインインを調査し、ユーザーのリスクプロファイルにアクセスし、デバイスのコンプライアンスを特定し、分析結果のレポートを作成する
